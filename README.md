### Let
  1. Let declared variables are valid only within the code block in which they are located
    ```
    if (true) {
      let a = 10
    }
    console.log(a) // a is not defined

  ```
  
  
  2. Let does not have variable promotion
    ```
    console.log(a)  // a is not deined
    let a = 10
    ```
